# Spring Login

로그인과 관련된 여러 문제를 해결하는 Spring boot 프로젝트

## 요구사항

### 로그인 기능 구현

- Remember me 기능 구현
  - 메모리
  - db
- 구현 방법
  - 쿠키
  - 세션
  - Spring security
  - JWT
  - OAuth

### 로그아웃 기능 구현

- 로그아웃 시 홈페이지 이동

### 회원가입 기능 구현

- 중복 회원 검증
- 비밀번호 암호화
- 이메일 인증

### 유저 별 권한 부여

- 관리자, 일반 유저 권한 부여

### 타임리프를 활용한 페이지 구현

- 홈페이지
  - 로그인 페이지로 이동 가능
  - 회원가입 페이지로 이동 가능
  - 로그인 시, 사용자의 이메일이 노출되어야 함
  - 로그인 시, 로그아웃 버튼이 노출되어야 함
- 로그인 페이지
  - 사용자의 이메일, 비밀번호를 입력받음
- 회원가입 페이지
  - 사용자의 이메일, 비밀번호를 입력받음
